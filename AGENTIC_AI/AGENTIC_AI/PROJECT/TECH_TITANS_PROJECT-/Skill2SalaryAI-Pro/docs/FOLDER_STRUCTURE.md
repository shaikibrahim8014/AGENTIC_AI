## What Each Part Does

### `app.py`

Backend server for:

- Local user signup and login
- SQLite database storage
- Salary ML model training
- Salary prediction API
- What-If Analysis API
- SalarAI Guide chatbot API

### `streamlit_app.py`

Streamlit deployment app for:

- Resume AI
- Salary prediction
- All individual model predictions
- Charts and What-If Analysis
- SalarAI Guide chatbot

### `index.html`

Main website layout:

- Login page
- Dashboard
- Resume AI
- Salary Lab
- Career Roadmap
- SalarAI Guide chatbot markup

### `assets/app.js`

Frontend logic:

- Login and signup calls
- Resume AI analysis
- Salary form submission
- Model prediction cards
- Charts
- What-If Analysis
- Chatbot UI behavior
- Animated network canvas

### `assets/style.css`

Website design:

- Modern glass UI
- Professional animated background
- Responsive layout
- Salary cards
- Chatbot styling

### `docs/`

Complete project documentation for setup, APIs, ML model behavior, user guide, and folder structure.
