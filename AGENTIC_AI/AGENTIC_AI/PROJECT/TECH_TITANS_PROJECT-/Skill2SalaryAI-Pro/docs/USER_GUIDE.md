# User Guide

## 1. Create Account

Open the website and choose `Create account`. Enter a username and password.

## 2. Analyze Resume

Go to Resume AI, paste resume text, and click `Analyze resume`.

The app shows:

- ATS score
- Role match percentage
- Detected skills
- Priority improvements

## 3. Predict Salary

Go to Salary Lab and fill:

- Target role
- Experience
- Education
- City category
- Industry
- Current salary in LPA
- Current skills

Click `Generate salary prediction`.

The app shows:

- Final ensemble salary in Indian LPA
- Salary range
- Confidence score
- Individual model predictions
- Career insights
- Skills to increase salary

## 4. What-If Analysis

After prediction, select new skills you want to learn. Click `Calculate future salary` to compare current salary and future salary.

## 5. SalarAI Guide

Use the bottom-right `SalarAI Guide` chatbot to ask questions about:

- Salary prediction
- Resume AI
- ML models
- Charts
- Login and setup
- Website features
